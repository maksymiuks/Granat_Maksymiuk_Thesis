# Structure

Structure of the repo

* data 
  * annotations - catalog with .rds files containing preprcoessed annotations results
  * phones - catalogs with .csv file that is Phones dataset
* plots - catalog with plots used in the thesis
* python_scraping - catalog with ipynotebook that we use to scrap phones from the website
* questionnaire - catalog with survey shiny app
* results - catalog with .rds files containing experiments results
* scripts - catalog contaiining scripts that were used in experiments for instance model creation and measures
* shinyforms_fork - shinyforms R pacakge edited according to our needs
* trash - files that are no longer in use
